Intro

Op deze pagina leggen wij c++ of beter bekend als de code taal van arduino uit.

Basis

Elke code van C++ bestaat uit 3 hoofddelen:

1. Definities: Hier defininieer je alle variabel, in- en uitgangen en je gebruikte bibliotheken.

2. Setup: Hier komt de code die 1 keer wordt uitgevoerd.

3. Loop: De code die in dit gedeelte staat wordt constant herhaald.


        led = 2;
        void setup () {
          pinMode(led, INPUT);
        }
  
        void loop () {
          digitalWrite(led, HIGH);
          delay(10);
          digitalWrite(led, LOW);
          delay(10);
        }
