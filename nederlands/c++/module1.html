Basis

Elke code van C++ bestaat uit 3 hoofddelen:

1. Definities: Hier defininieer je alle variabel, in- en uitgangen en je gebruikte bibliotheken.

2. Setup: Hier komt de code die 1 keer wordt uitgevoerd.

3. Loop: De code die in dit gedeelte staat wordt constant herhaald.


          led = 2; // hier definieer je een op welke plaats een led is aangesloten
          
          void setup () {
            pinMode(led, OUTPUT); // hier zeg je of de led een input of een output is
          }
    
          void loop () {
            // hieronder een korte code die een led laat knipperen
            digitalWrite(led, HIGH); // led aan
            delay(10); // een wacht periode
            digitalWrite(led, LOW); // led uit
            delay(10); // wacht periode
          }
